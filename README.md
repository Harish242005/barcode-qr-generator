# 👑 Elegant QR Creator

A simple and elegant Streamlit app to upload images, create public links, and generate QR codes.  
Also supports barcodes, batch processing, QR decoding, and more.

---

## ✨ Features

- 🖼️ Upload image and create public link  
- 🔗 Convert image link to QR code  
- 🔳 Generate QR from text or URL  
- 🏷️ Generate barcode from numbers  
- 📂 Batch CSV upload for QR / barcode  
- 👤 Contact card QR  
- 🎫 Event ticket QR  
- 📍 Location QR  
- 🔍 QR code decoder  
- 📜 History tracking  

---

## 🚀 How it works

1. Upload an image  
2. Image is uploaded to Cloudinary  
3. A public link is created  
4. QR code is generated from that link  
5. Scan QR to open the image
